# OQS Dart - Claude Code Configuration

## Overview

This is a **security-audited fork** of [oqs](https://pub.dev/packages/oqs).
### Why a Fork?

1. **Security Control** - We audit all upstream changes before merging to prevent supply chain attacks or backdoors in cryptographic code
2. **Pre-built Binaries** - We build and ship native liboqs libraries for all platforms via GitHub Actions
3. **Version Pinning** - We control exactly which liboqs version is used across all platforms

### Upstream Repository

- **Original package**: https://pub.dev/packages/oqs
- **Original repo**: https://github.com/bardiakz/oqs
- **liboqs**: https://github.com/open-quantum-safe/liboqs

## Native Library Build System

### How It Works

1. Version is stored in `LIBOQS_VERSION` file
2. GitHub Actions workflow builds liboqs for all platforms when this file changes
3. Built libraries are committed to `bin/` directory
4. Libraries are included in the package for Flutter/Dart apps

### Updating liboqs Version

```bash
# Change version and push - CI will build automatically
echo "0.16.0" > LIBOQS_VERSION
git add LIBOQS_VERSION
git commit -m "Update liboqs to 0.16.0"
git push
```

### Manual Build Trigger

Go to Actions -> "Build liboqs Native Libraries" -> Run workflow

## Security Considerations

### Supply Chain Security

- **DO NOT** merge upstream changes without security review
- All native libraries are built from source in GitHub Actions (auditable)
- Pin to specific liboqs releases (no `main` branch builds)

### Code Review Checklist

When merging upstream changes:
1. Review all Dart code changes for backdoors
2. Check FFI bindings for memory safety issues
3. Verify no hardcoded keys or suspicious network calls
4. Ensure cryptographic operations use proper randomness

## Integration with Enigma

This package provides post-quantum cryptography for Enigma:
- **ML-KEM** (Kyber) - Key encapsulation for secure key exchange
- **ML-DSA** (Dilithium) - Digital signatures for message authentication
- **OQSRandom** - Cryptographically secure random number generation

Used in:
- `flutter/packages/enigma_app/lib/core/crypto/` - Cryptographic operations
- Key exchange during session establishment
- Message signing and verification
